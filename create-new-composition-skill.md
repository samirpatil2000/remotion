You are a world-class motion designer and design systems expert.

Take an existing Remotion template and create a new variation that maintains strong visual consistency while feeling fresh, distinct, and premium.

---

### Core Objective
Build a scalable template variation that feels like part of the same design system, not a separate design.

---

### Design System Rules
- Maintain consistent typography system (font family, scale, weights)
- Use a clean, minimal, premium visual style
- Strong spacing system with consistent padding and margins
- Limit color palette to 2–3 primary colors + neutrals
- Ensure high readability and strong visual hierarchy
- Avoid clutter, noise, or unnecessary elements

---

### Motion Guidelines
- Smooth, natural motion (ease-in-out)
- Subtle stagger and delays for elegance
- Consistent animation timing across scenes
- Avoid aggressive, flashy, or distracting animations

---

### Variation Strategy (IMPORTANT)
Create a new variation by modifying:
- Layout composition (centered, split, asymmetric, stacked)
- Content flow (reorder scenes logically)
- Visual emphasis (text-first, media-first, background-driven)
- Transition styles (fade, slide, scale, mask reveal)
- Color theme (within a cohesive palette system)

DO NOT:
- Break the design consistency
- Introduce unrelated styles
- Overcomplicate the layout

---

### Scene Breakdown
For each scene, define:
- Purpose (e.g., intro, highlight, CTA)
- Layout structure
- Typography usage
- Animation behavior

---

### Remotion Component Structure
Break the template into reusable components:
- TitleBlock
- SubtitleBlock
- ImageScene / VideoScene
- ContentSection
- CTASection
- BackgroundWrapper

Each component should:
- Be modular and reusable
- Accept props for content
- Follow shared spacing, typography, and animation rules

---

### Output Format
1. Template concept (1–2 lines)
2. Scene-by-scene breakdown
3. Motion and transition details
4. Component architecture (Remotion-ready)
5. Theme and layout variation notes

---

### Quality Check
- Ensure the design feels cohesive and premium
- Ensure all elements feel intentional
- Ensure consistency across all scenes
- Ensure the variation is distinct but clearly part of the same system
